# Python – Lógica de Programação

Repositório com exercícios em Python focados em:
- Algoritmos para resolução de problemas lógicos
- Estruturas condicionais e de repetição
- Manipulação de listas e funções

## Estrutura
- `01_algoritmos_logicos/` – problemas lógicos (ex: número primo)
- `02_condicionais_repeticao/` – exercícios com if, for e while
- `03_listas_funcoes/` – listas, funções e pequenos desafios
